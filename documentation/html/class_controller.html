Detailed Description

Controller class responsible for controlling the simmulation.

This class is responsible for the simulation. It holds every information needed using the Model objects, and a pathfinding algorithm is implemented in this class used for finding the most optimal path for the robots used in the simulation. Also uses a signal which is connected to a handler in the EventManager class, which sends the updated state to the clients every tick.
