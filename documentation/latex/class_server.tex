\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_server}{Server}} class implementing a TCP server. 

This is the class which is instantiated by the entry point of the program (the main class) This class implements a TCP server, and holds the \mbox{\hyperlink{class_event_manager}{Event\+Manager}} and \mbox{\hyperlink{class_controller}{Controller}} objects On creation, the class starts to listen on the default public IP address and on port 1432.
