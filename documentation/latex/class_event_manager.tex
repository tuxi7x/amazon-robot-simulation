\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_event_manager}{Event\+Manager}} Class for managing event-\/based communication with the clients. 

This class holds all the connected clients and listens for incoming messages, and when processesed, it interacts with the \mbox{\hyperlink{class_controller}{Controller}} object. Also this class is responsible for sending the messages to every client.
