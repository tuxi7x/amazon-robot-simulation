The \mbox{\hyperlink{class_editor_grid_button}{Editor\+Grid\+Button}} class is a special QPush\+Button that is only used in the grid of the Editor window. It is mainly used to correctly handle drag and drop actions, but it implements switching stylesheets of this button. It stores the row of the column of this button in the editor grid.
